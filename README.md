# CI/CD Next.js

Projeto com pipeline de Integração Contínua e Deploy Contínuo (CI/CD) usando GitHub Actions e Vercel.

## 🌐 Site Publicado

[https://ci-cd-next-steel.vercel.app](https://ci-cd-next-steel.vercel.app)

## 📦 Repositório GitHub

[https://github.com/Danielmsouza-dev/ci-cd-next](https://github.com/Danielmsouza-dev/ci-cd-next)

## 🚀 O que esse projeto faz?

- Executa lint, testes e build a cada push na branch `main`.
- Se tudo passar, faz o deploy automático na Vercel.
- A página inicial exibe "Portal de Viagens" e tem uma rota `/destinos` com uma listagem simples.
